# ⏰ Digital Clock & Countdown Timer

A stylish and functional web-based time management tool. This project combines a real-time digital clock with a custom countdown timer, all wrapped in a modern, responsive **Glassmorphism** interface.

---

## ✨ Features
* 🕒 **Real-time Clock:** Displays current time (hours, minutes, seconds) and date accurately.
* ⏲️ **Custom Timer:** Set a specific countdown in minutes and seconds with alert notification.
* 💎 **Glassmorphism UI:** Stunning transparent design with blur effects and neon accents.
* 📱 **Fully Responsive:** Optimized for a seamless experience on mobile, tablet, and desktop.
* 🎮 **Easy Controls:** Simple Start, Stop, and Reset functions for the timer.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Advanced styling, gradients, and media queries for responsiveness.
* **JavaScript (Vanilla):** Logic for time calculation, intervals, and UI updates.

## ⚙️ How to Use
1. **Clock:** Automatically shows your local time upon loading.
2. **Timer:** - Enter minutes and seconds in the input boxes.
   - Click **Start** to begin the countdown.
   - Use **Stop** to pause and **Reset** to clear the timer.
